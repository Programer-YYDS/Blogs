# 杜少雄的个人博客代码托管仓库

### 项目已成功部署至阿里云轻量级服务器
### 访问地址 : www.ShaoxiongDu.top

### 部署教程
####  1. 将项目克隆至本地
####  2. 修改配置文件中的数据库信息
####  3. 修改配置文件中的ddl为create
####  4. 运行程序,自动创建表结构
####  5. 手动在user表中插入一条数据 (注意密码存储采用MD5加密)
####  6. local host:80即可访问首页 localhost:80/admin 访问后台登陆页面
##### 注:  如果要修改端口号  主配置文件即可修改
